## Extras

### Benchmarking solutions

There are package scripts for running [hyperfine](https://github.com/sharkdp/hyperfine), which you'll need to [install](https://github.com/sharkdp/hyperfine#installation) separately.

```
bun bench ./01
```

```
bun bench:all
```
